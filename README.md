# 42sh
A C shell based on TCSH

```
$ git clone https://github.com/pacome35220/my_chat.git
$ cd 42sh
$ make
$ ./42sh
```

## Features
- Separator ; || &&
- Multiples pipes
- Redirections (< / << / > / >>)
- Parenthesis
- Environment variables
- ~ with users
- Globbing (*)
- Scripting
- Prompt

## Builtins
- alias
- builtins
- cd
- chdir
- echo
- env
- exit
- getenv
- kill
- power4
- setenv
- snake
- unsetenv
- where
- which
- whoami

## Games (bonus)
- snake (move with arrows)
- power4 ('l' to move left, 'r' to move right, 'd' to valid, 'q' to quit)
